Sachan 是一个多功能的 Telegram 机器人。详细信息请参见 [帮助](./src/help.js)。

> 官方机器人 [@sachanbot](https://t.me/sachanbot) 目前处于离线状态。

## 安装

- npm: `npm i sachan-bot`
- Yarn: `yarn add sachan-bot`

## 使用方法

要启动机器人，你需要提供以下环境变量：

- `TELEGRAM_BOT_TOKEN`
- [`DEEPAI_API_KEY`](https://deepai.org/) (可选，如果你需要 waifu2x 功能)

## 开发

- 安装依赖 `yarn`
- 启动 `node index.js`
